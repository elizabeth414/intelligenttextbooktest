# Intelligent Textbook

A documentation site built with [MkDocs](https://www.mkdocs.org) and the [Material theme](https://squidfunk.github.io/mkdocs-material/), published via GitHub Pages.

## Live Site

[https://elizabeth414.github.io/intelligenttextbooktest/](https://elizabeth414.github.io/intelligenttextbooktest/)

## Getting Started

### Prerequisites

- [Miniconda](https://docs.conda.io/en/latest/miniconda.html)

### Setup

```bash
conda activate mkdocs
```

### Local Development

```bash
mkdocs serve
```

Then open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.

### Publishing

```bash
mkdocs gh-deploy
```

## Project Structure
